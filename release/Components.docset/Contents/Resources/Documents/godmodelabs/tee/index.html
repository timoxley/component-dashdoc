tee

tee(1)

Usage

var tee = require('tee');

src
  .pipe(tee(
    dest1,
    dest2,
    dest3
  ))
  .pipe(somewhereElse)

Installation

$ npm install tee-1



or


$ component install godmodelabs/tee

API

tee(destination, ...)

Creates a new through Stream that pipes all incoming data to each
destination-Stream.
