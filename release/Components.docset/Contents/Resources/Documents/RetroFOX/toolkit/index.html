toolkit

Build a toolkit object container.

Installation

$ component install retrofox/toolkit

Example

Load component

var toolkit = require('toolkit');

Create array toolkit adding map and to-object array components.

var array = toolkit(require, ['map', 'to-object']);

Also add select component using .add() method.

array.add('select');

// Using array toolkit

users = array.select(users, function(user){
  return user.age > 20;
});
