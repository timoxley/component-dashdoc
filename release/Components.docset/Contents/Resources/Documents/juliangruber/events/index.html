events

Node's events module in the browser.

This is a wrapper around component/emitter that is api-compatible with node's events module.

Events Node.js v0.8.11 Manual & Documentation

Installation

Install with component(1)

$ npm install juliangruber/events

or

$ npm install events-component

Usage example

var EventEmitter = require('events').EventEmitter;
var ee = new EventEmitter();

ee.on('foo', function(data) {
  console.log('foo received', data);
})

ee.emit('foo', 'bar');
// -> 'foo received bar'
