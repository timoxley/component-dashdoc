ts

Humanify unix timestamps

Usage

$ ts 1352700000
Mon Nov 12 2012 07:00:00 GMT+0100 (CET) (6h 45m 24s ago)

var ts = require('ts')

ts(1352700000) == 'Mon Nov 12 2012 07:00:00 GMT+0100 (CET) (6h 45m 24s ago)'

Installation

$ npm install -g ts



or


$ npm install ts
