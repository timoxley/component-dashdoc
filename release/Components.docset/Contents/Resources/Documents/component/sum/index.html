sum

Sum utility

Installation

$ component install component/sum

API

sum(array, [fn|string])

Sum an array of numbers:

sum([1,2,3,4,5]);

Sum properties with callback:

var sum = sum(users, function(user){
  return user.age;
});

map(array, string)

Sum properties in string:

sum(users, 'age');
sum(repos, 'stats.watchers');
